Installation Guide

Rock Band 4 Deluxe requires a copy of vanilla Rock Band 4 in order to run. We hope this is clear.

  • PS4: CUSA02084 (USA) or CUSA02901 (EU) version 2.21 (2.3.7 in game)
  • ShadPS4: CUSA02084 (USA) or CUSA02901 (EU) version 1.08 (1.3.0 in game)

Installing on PC, via ShadPS4*

*RB4DX works on Windows and Linux versions of ShadPS4.

Download Rock Band 4 Deluxe if you haven't already

  • Extract the .zip file.
  • Drag and drop the GoldHEN folder into ShadPS4's user/data folder in a subfolder titled CUSA02084 or CUSA02901, depending on your base games Title ID.
  • Then take the RB4DX-Plugin.prx file in the GoldHEN/plugins folder into the game files in a subfolder titled sce_module.
  • Overwite any existing files.
    • If Rock Band 4 refuses to start after installing RB4DX, try re-extracting the .zip file with a different program.
    • If Rock Band 4 is loading, but no Deluxe features are showing up, you might have to rename the plugin file to zRB4DX-Plugin.prx (if the plugin and RB4DX-1.08 folder are in the right locations).
    • If some Rock Band 4 Deluxe features are not functioning properly, you need to copy PS4 sysmodules into a folder called sys_modules into ShadPS4's user folder.

✅ Rock Band 4 Deluxe is now installed and ready to play!

Rock on!

Installing on PS4*

*RB4DX requires a modded PS4 console (GoldHEN) in order to run.

Download Rock Band 4 Deluxe if you haven't already

  • Extract the .zip file.
  • Copy the GoldHEN folder to the /data/ folder on your PS4, either via FTP or copying it to a flash drive and using a file browser on the PS4.
  • Overwite any existing files.
    • If the plugin is not loading, you may need to change the file permissions of the plugin to allow it to run. This can be done with the PS4/GoldHEN Cheats Manager, in the settings there is a button labelled `Set file permissions for Plugins`, pressing this should make the plugin load correctly.
    • If the plugin is loading, but no dx modifications are showing up, then you may need to change the permissions of the RB4DX folder on the PS4, or retransfer the RB4DX folder entirely.

✅ Rock Band 4 Deluxe is now installed and ready to play!

You can play the game using the original disc or via a backup on your hard drive. Rock on!
